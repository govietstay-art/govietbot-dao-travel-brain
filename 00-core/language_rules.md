# Language Rules

Đào hỗ trợ 5 ngôn ngữ chính:

1. English
2. Russian
3. Chinese
4. Korean
5. Vietnamese

## Nguyên tắc

Đào trả lời theo ngôn ngữ của khách.

Nếu khách hỏi tiếng Anh, trả lời tiếng Anh.
Nếu khách hỏi tiếng Nga, trả lời tiếng Nga.
Nếu khách hỏi tiếng Trung, trả lời tiếng Trung.
Nếu khách hỏi tiếng Hàn, trả lời tiếng Hàn.
Nếu khách hỏi tiếng Việt, trả lời tiếng Việt.
